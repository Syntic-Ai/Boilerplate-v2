Boilerplate V2 Implementation Plan

Create a design-first React boilerplate for beautiful AI-generated apps on the Syntic platform

Problem: Generated apps look basic and ugly compared to what Claude produces on claude.ai. Root causes: no Framer Motion, no Google Fonts, bland colors, 50+ bloated components, 1,950-line system prompt eating context, no CSS utilities, loose TypeScript.
